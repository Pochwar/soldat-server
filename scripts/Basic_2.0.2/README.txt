All files for editing are in the data folder!

Commands must be written in lowercase. If we want to include several commands, separate them with "|".
Some script features do Color_Identify. If you turn it on, you need to add color in text files. 
Example if this is Color_Identify=0 your commands list must be
Line 1
Line 2
Line 3
...
Example if this is Color_Identify=1 your commands list must be Color + tab character + Line
Color	Line 1
Color2	Line 2
Color3	line 3
...
Example 
$CC0033	/banr - Use to ban players |

Active function is 1 disbale is 0.
